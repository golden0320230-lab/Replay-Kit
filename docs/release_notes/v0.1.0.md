# ReplayKit v0.1.0

Initial public release of ReplayKit.

## Highlights

- Deterministic artifact format with schema validation and migration support.
- Capture boundaries for model/tool/http events with redaction defaults.
- Offline replay (stub + hybrid) with first-divergence diffing.
- CLI workflows for:
  - `record`, `replay`, `diff`, `assert`
  - `bundle`, `verify`, `snapshot`, `benchmark`, `migrate`, `ui`
- Cross-platform CI coverage and deterministic replay assertions.

## Security

- Safe-by-default redaction for common secret-bearing fields.
- Artifact signature verification workflow for integrity checks.

## Notes

- Python 3.10+ is required.
- See `CHANGELOG.md` for detailed change history.
